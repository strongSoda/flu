## Flu

Intended to be a website generator cli.

### Usage

- `$ pip install flu`
- `$ flu`

### Local Development
- Clone `$ git clone https://github.com/strongSoda/flu.git`

- Create a venv (optional)

- `$ pip install -r requirements.txt`

- `$ python flu.py --help`

### Tools

Written in [click](https://github.com/pallets/click)
